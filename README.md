## Clone this project

```bash
> git clone https://github.com/rizkiramadhan4617/RR018-BOT
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd RR018-BOT
> bash install.sh
```

### Usage
```bash
> npm start
```
